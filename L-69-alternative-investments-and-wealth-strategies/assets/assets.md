# Asset Specifications for L-69: Alternative Investments and Wealth Strategies

This chapter includes 8 HTML assets for the skill builder and Day 2 activities.

## Day 1 Skill Builder

### 1. Alternative Investment Portfolio Analyzer
- **Purpose:** Primary skill builder for analyzing alternative investment options
- **Format:** Interactive web application
- **HTML File:** Alternative_Investment_Evaluator.html

## Day 2 Activity Assets

### 2. Asset Correlation Matrix Tool
- **Purpose:** Compare how different asset pairs move together using 2010-2023 historical data
- **Format:** Interactive analysis/visualization tool
- **HTML File:** Asset_Correlation_Matrix.html
- **Supports Activity:** Activity 1 - Asset Correlation Explorer

### 3. Real Estate Investment Calculator
- **Purpose:** Compare direct property ownership vs. REIT investment with 10-year calculations and scenario variations
- **Format:** Interactive calculator
- **HTML File:** Real_Estate_Investment_Calculator.html
- **Supports Activity:** Activity 2 - Real Estate Investment Comparison

### 4. Cryptocurrency Risk Simulator
- **Purpose:** Experience cryptocurrency volatility through historical periods (2017-2024) with allocation scenarios
- **Format:** Interactive simulation
- **HTML File:** Cryptocurrency_Risk_Simulator.html
- **Supports Activity:** Activity 3 - Cryptocurrency Risk Simulation

### 5. Portfolio Diversification Visualizer
- **Purpose:** Create and compare three portfolio strategies for a 22-year-old with scenario testing
- **Format:** Interactive builder/visualizer
- **HTML File:** Portfolio_Diversification_Visualizer.html
- **Supports Activity:** Activity 4 - Portfolio Diversification Strategy Builder

### 6. Alternative Investment Decision Framework Checklist
- **Purpose:** Establish personal rules for approaching alternative investments with checklist completion
- **Format:** Interactive checklist/planning tool
- **HTML File:** Alternative_Investment_Decision_Framework.html
- **Supports Activity:** Activity 5 - Alternative Investment Decision Framework
