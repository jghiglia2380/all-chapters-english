# Asset Specifications for L-61: Contract Evaluation and Consumer Protection

## Overview
This chapter includes 8 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Contract Analyzer
- **Purpose:** Primary skill builder for systematic contract analysis and evaluation
- **Format:** Interactive web application
- **HTML File:** Consumer_Protection_Resource_Finder.html

---

## Day 2 Activity Assets

### 2. Real Contract Deep-Dive Analyzer
- **Purpose:** Analyze three complete real-world contracts (apartment lease, gym membership, cell phone service) using systematic evaluation framework
- **Format:** Interactive web application
- **HTML File:** Real_Contract_Analyzer.html
- **Supports Activity:** Activity 1 - Real Contract Deep-Dive Analysis

### 3. Consumer Rights Database
- **Purpose:** Research applicable consumer protection laws for various scenarios (bait-and-switch, defective products, forced arbitration, cooling-off periods)
- **Format:** Interactive database/reference tool
- **HTML File:** Consumer_Rights_Database.html
- **Supports Activity:** Activity 2 - Consumer Rights Application

### 4. Red Flag Identifier Tool
- **Purpose:** Review and classify contract excerpt red flags as Acceptable, Concerning, or Unacceptable across three rounds
- **Format:** Interactive web application
- **HTML File:** Red_Flag_Identifier.html
- **Supports Activity:** Activity 3 - Red Flag Identification Challenge

### 5. Negotiation Strategy Guide
- **Purpose:** Develop negotiation approaches and scripts for problematic contract terms
- **Format:** Interactive worksheet/guide builder
- **HTML File:** Negotiation_Strategy_Builder.html
- **Supports Activity:** Activity 4 - Negotiation Strategy Workshop

### 6. Personal Contract Evaluation Planner
- **Purpose:** Create action plans for current and anticipated contracts with specific review and negotiation strategies
- **Format:** Interactive planning tool
- **HTML File:** Personal_Contract_Evaluation_Planner.html
- **Supports Activity:** Activity 5 - Personal Contract Evaluation

---

## Supporting Materials

### 7. Reference Sheet
- **Purpose:** Quick reference for consumer protection laws and key concepts
- **Format:** 1-page printable
- **HTML File:** Reference_Sheet.html

### 8. Supporting Worksheet
- **Purpose:** Practice exercises for contract evaluation and consumer rights application
- **Format:** 2-page printable
- **HTML File:** Supporting_Worksheet.html
