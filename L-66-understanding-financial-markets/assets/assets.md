# Asset Specifications for L-66: Understanding Financial Markets

## Overview
This chapter includes 8 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Market Trading Simulator
- **Purpose:** Primary skill builder for understanding market mechanics and order execution
- **Format:** Interactive web application
- **HTML File:** Market_Trading_Simulator.html

---

## Day 2 Activity Assets

### 2. Stock Quote Analysis Worksheet
- **Purpose:** Record and analyze stock quotes for three different securities (large-cap, small-cap, ETF) with spread and volume analysis
- **Format:** Interactive worksheet/calculator
- **HTML File:** Stock_Quote_Analysis_Worksheet.html
- **Supports Activity:** Activity 1 - Quote Reading and Interpretation Challenge

### 3. Order Type Practice Simulator
- **Purpose:** Experience how different order types (market, limit) execute through five scenarios
- **Format:** Interactive simulation
- **HTML File:** Order_Type_Practice_Simulator.html
- **Supports Activity:** Activity 2 - Market Trading Simulator - Order Type Practice

### 4. Trading Cost Calculator
- **Purpose:** Calculate single trade costs, annual costs for different investor profiles, and lifetime impact analysis
- **Format:** Interactive calculator/spreadsheet
- **HTML File:** Trading_Cost_Calculator.html
- **Supports Activity:** Activity 3 - Trading Cost Analysis and Strategy Comparison

### 5. Brokerage Platform Comparison Tool
- **Purpose:** Explore and compare brokerage platforms with feature matrix and cost analysis
- **Format:** Interactive comparison tool
- **HTML File:** Brokerage_Platform_Comparison.html
- **Supports Activity:** Activity 4 - Brokerage Account Exploration

### 6. Personal Trading Strategy Builder
- **Purpose:** Create personalized trading strategy with investor profile, order type rules, and first trade plan
- **Format:** Interactive planning tool
- **HTML File:** Personal_Trading_Strategy_Builder.html
- **Supports Activity:** Activity 5 - Personal Trading Strategy Development

---

## Supporting Materials

### 7. Reference Sheet
- **Purpose:** Quick reference for order types, market mechanics, and trading terminology
- **Format:** Multi-page printable
- **HTML File:** Reference_Sheet.html

### 8. Supporting Worksheet
- **Purpose:** Practice exercises for quote analysis and cost calculations
- **Format:** Multi-page printable
- **HTML File:** Supporting_Worksheet.html
