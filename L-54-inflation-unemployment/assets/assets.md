# Asset Specifications for L-54: Inflation, Unemployment, and Personal Finance

## Overview
This chapter includes 7 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Inflation Impact Calculator
- **Purpose:** Calculate purchasing power erosion and real vs. nominal values for personal finance decisions
- **Format:** Interactive web application
- **HTML File:** Inflation_Impact_Calculator.html

---

## Day 2 Activity Assets

### 2. Real vs. Nominal Value Calculator
- **Purpose:** Calculate purchasing power erosion over time at different inflation rates and salary adjustments
- **Format:** Interactive calculator
- **HTML File:** Real_Nominal_Calculator.html
- **Supports Activity:** Activity 1 - Real vs. Nominal Value Calculations

### 3. Inflation Protection Strategy Planner
- **Purpose:** Plan investment strategies, wage negotiation approaches, and spending adjustments for inflation protection
- **Format:** Interactive planning tool
- **HTML File:** Inflation_Protection_Planner.html
- **Supports Activity:** Activity 2 - Inflation Protection Strategies

### 4. Unemployment Navigation Tool
- **Purpose:** Assess job market strategies for frictional, structural, and cyclical unemployment scenarios
- **Format:** Interactive assessment/planner
- **HTML File:** Unemployment_Navigation_Tool.html
- **Supports Activity:** Activity 3 - Unemployment Navigation Strategies

### 5. Economic Scenario Analyzer
- **Purpose:** Allocate funds across different economic scenarios (high inflation/low unemployment, stagflation, etc.)
- **Format:** Interactive decision tool
- **HTML File:** Economic_Scenario_Analyzer.html
- **Supports Activity:** Activity 4 - Economic Scenario Analysis

### 6. Reference Sheet
- **Purpose:** Quick reference for inflation and unemployment concepts
- **Format:** 1-page printable
- **HTML File:** Reference_Sheet.html

### 7. Supporting Worksheet
- **Purpose:** Practice exercises for inflation and unemployment analysis
- **Format:** 2-page printable
- **HTML File:** Supporting_Worksheet.html
