# Asset Specifications for L-60: Understanding Financial Contracts

## Overview
This chapter includes 8 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Contract Analysis Tool
- **Purpose:** Interactive tool to analyze contracts, identify essential elements, and spot red flags
- **Format:** Interactive web application
- **HTML File:** Contract_Review_Checklist.html

---

## Day 2 Activity Assets

### 2. Contract Elements Finder Tool
- **Purpose:** Rapidly identify essential contract elements in various real-world contracts through timed scavenger hunt
- **Format:** Interactive web application
- **HTML File:** Contract_Elements_Finder.html
- **Supports Activity:** Activity 1 - Contract Elements Scavenger Hunt

### 3. Fine Print Analyzer Tool
- **Purpose:** Identify problematic clauses, hidden fees, and red flags in contract fine print across three rounds
- **Format:** Interactive web application
- **HTML File:** Fine_Print_Analyzer.html
- **Supports Activity:** Activity 2 - The Fine Print Challenge

### 4. Contract Comparison Tool
- **Purpose:** Compare competing contracts (apartments, auto loans, cell phone plans) to identify best financial choice
- **Format:** Interactive calculator/comparison tool
- **HTML File:** Contract_Comparison_Tool.html
- **Supports Activity:** Activity 3 - Contract Comparison Station

### 5. Negotiation Scenario Cards
- **Purpose:** Role-play cards with contract negotiation scenarios, talking points, and objectives
- **Format:** Printable worksheet/cards
- **HTML File:** Negotiation_Scenario_Cards.html
- **Supports Activity:** Activity 4 - Contract Negotiation Role-Play

### 6. Contract Safety Checklist Builder
- **Purpose:** Create personalized checklist for evaluating future contracts with customizable questions
- **Format:** Interactive web application
- **HTML File:** Contract_Safety_Checklist_Builder.html
- **Supports Activity:** Activity 5 - Personal Contract Safety Checklist Builder

---

## Supporting Materials

### 7. Reference Sheet
- **Purpose:** Quick reference for key concepts
- **Format:** 1-page printable
- **HTML File:** Reference_Sheet.html

### 8. Supporting Worksheet
- **Purpose:** Practice exercises for contract element identification, fee calculations, red flag spotting
- **Format:** 2-page printable
- **HTML File:** Supporting_Worksheet.html
