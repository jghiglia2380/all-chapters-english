# Asset Specifications for L-47: Introduction to Investment Types

## Overview
This chapter includes 7 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Investment Type Analyzer
- **Purpose:** Main interactive tool for exploring investment types, comparing scenarios, and building sample portfolios
- **Format:** Interactive web application
- **HTML File:** Investment_Type_Analyzer.html

---

## Day 2 Activity Assets

### 2. Investment Characteristics Comparison Chart
- **Purpose:** Rate and compare investment types (stocks, bonds, mutual funds, ETFs) across potential return, risk level, liquidity, complexity, minimum investment, and income generation
- **Format:** Interactive worksheet with rating scales
- **HTML File:** Investment_Characteristics_Chart.html
- **Supports Activity:** Activity 1 - Investment Type Characteristics Deep Dive (15 minutes)

### 3. Stock vs. Bond Performance Simulator
- **Purpose:** Simulate investment performance under different economic scenarios (strong growth, recession, high inflation, rising rates) and compare 30-year historical returns
- **Format:** Interactive simulator with historical data
- **HTML File:** Performance_Simulator.html
- **Supports Activity:** Activity 2 - Stock vs. Bond Performance Simulator (12 minutes)

### 4. Fee Impact Calculator
- **Purpose:** Calculate long-term cost differences between low-cost ETFs, average mutual funds, and high-cost active funds over 30-year periods
- **Format:** Interactive calculator with compound growth visualization
- **HTML File:** Fee_Impact_Calculator.html
- **Supports Activity:** Activity 3 - Mutual Fund vs. ETF Comparison Challenge (12 minutes)

### 5. Goal-Based Portfolio Builder
- **Purpose:** Create appropriate investment allocations for different financial scenarios (retirement, home down payment, mid-career) based on time horizon and risk tolerance
- **Format:** Interactive portfolio designer with allocation visualization
- **HTML File:** Portfolio_Builder.html
- **Supports Activity:** Activity 4 - Goal-Based Portfolio Builder (11 minutes)

### 6. Investment Decision Framework
- **Purpose:** Personal investment profile assessment and decision tree to determine recommended allocation and specific investment types
- **Format:** Interactive self-assessment with recommendations
- **HTML File:** Investment_Decision_Framework.html
- **Supports Activity:** Activity 5 - Investment Type Decision Framework (10 minutes)

### 7. Investment Types Reference Sheet
- **Purpose:** Quick reference for all investment types with characteristics, risk/return profiles, and key terms
- **Format:** Printable reference document
- **HTML File:** Investment_Types_Reference_Sheet.html
- **Supports Activity:** All activities - provides reference information
