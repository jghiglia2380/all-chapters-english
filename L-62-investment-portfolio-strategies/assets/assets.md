# Asset Specifications for L-62: Investment Portfolio Strategies

## Overview
This chapter includes 8 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Advanced Portfolio Builder
- **Purpose:** Primary skill builder for portfolio construction and strategy development
- **Format:** Interactive web application
- **HTML File:** Portfolio_Builder_Tool.html

---

## Day 2 Activity Assets

### 2. Lifecycle Portfolio Constructor
- **Purpose:** Build three portfolios for different life stages (age 25, 45, 60) with projected returns and allocation justification
- **Format:** Interactive calculator/builder
- **HTML File:** Lifecycle_Portfolio_Constructor.html
- **Supports Activity:** Activity 1 - Lifecycle Portfolio Construction

### 3. Historical Scenario Tester
- **Purpose:** Test portfolios through historical market events (2008 crisis, 2020 COVID, 1970s stagflation) to understand resilience
- **Format:** Interactive simulation
- **HTML File:** Historical_Scenario_Tester.html
- **Supports Activity:** Activity 2 - Historical Scenario Testing

### 4. Rebalancing Simulator
- **Purpose:** Experience how rebalancing maintains allocation and potentially improves returns through multi-year simulation
- **Format:** Interactive simulation
- **HTML File:** Rebalancing_Simulator.html
- **Supports Activity:** Activity 3 - Rebalancing Simulation

### 5. Tax Efficiency Optimizer
- **Purpose:** Calculate 20-year after-tax outcomes for different asset location strategies
- **Format:** Interactive calculator
- **HTML File:** Tax_Efficiency_Optimizer.html
- **Supports Activity:** Activity 4 - Tax Optimization Strategy

### 6. Investment Policy Statement Builder
- **Purpose:** Create personalized investment plan with specific allocations, rebalancing rules, and triggers
- **Format:** Interactive planning tool
- **HTML File:** Investment_Policy_Statement_Builder.html
- **Supports Activity:** Activity 5 - Personal Investment Policy Statement

---

## Supporting Materials

### 7. Reference Sheet
- **Purpose:** Quick reference for portfolio strategy concepts and asset allocation guidelines
- **Format:** 1-page printable
- **HTML File:** Reference_Sheet.html

### 8. Supporting Worksheet
- **Purpose:** Practice exercises for portfolio construction and calculation
- **Format:** 2-page printable
- **HTML File:** Supporting_Worksheet.html
