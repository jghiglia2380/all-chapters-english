# Asset Specifications for L-59: Understanding Local Tax Structures

## Overview
This chapter includes 8 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Local Tax Calculator
- **Purpose:** Calculate and compare total local tax burdens across multiple jurisdictions
- **Format:** Interactive web application
- **HTML File:** Local_Tax_Calculator.html

---

## Day 2 Activity Assets

### 2. Property Tax Bill Analyzer
- **Purpose:** Decode property tax bills to understand components, taxing authorities, exemptions, and budget allocation
- **Format:** Interactive bill analysis tool
- **HTML File:** Property_Tax_Bill_Analyzer.html
- **Supports Activity:** Activity 1 - Property Tax Bill Deep Dive

### 3. Tax Jurisdiction Comparison Matrix
- **Purpose:** Compare multiple locations side-by-side for rent, sales tax, income tax, cost of living, and services
- **Format:** Interactive comparison tool
- **HTML File:** Tax_Jurisdiction_Comparison.html
- **Supports Activity:** Activity 2 - Location Comparison Challenge

### 4. Home Affordability Calculator with Tax Integration
- **Purpose:** Calculate complete housing costs including mortgage, property taxes, insurance, and PMI across jurisdictions
- **Format:** Interactive affordability calculator
- **HTML File:** Home_Affordability_Calculator.html
- **Supports Activity:** Activity 3 - Home Buying Tax Impact Simulation

### 5. Assessment Challenge Tool
- **Purpose:** Evaluate property assessments, compare with comparable sales, and determine grounds for appeal
- **Format:** Interactive assessment analysis tool
- **HTML File:** Assessment_Challenge_Tool.html
- **Supports Activity:** Activity 4 - Tax Assessment Challenge Station

### 6. Life Stage Tax Planner
- **Purpose:** Match life stage priorities with optimal location types and tax considerations
- **Format:** Interactive planning tool
- **HTML File:** Life_Stage_Tax_Planner.html
- **Supports Activity:** Activity 5 - Life Stage Tax Planning

### 7. Reference Sheet
- **Purpose:** Quick reference for property tax formula, millage rates, assessment ratios, local tax terminology
- **Format:** 1-page printable
- **HTML File:** Reference_Sheet.html

### 8. Supporting Worksheet
- **Purpose:** Tax calculation practice, jurisdiction comparison exercises, home affordability analysis
- **Format:** 2-page printable
- **HTML File:** Supporting_Worksheet.html
