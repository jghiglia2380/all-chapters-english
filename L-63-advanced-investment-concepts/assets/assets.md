# Asset Specifications for L-63: Advanced Investment Concepts

## Overview
This chapter includes 8 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Alternative Investment Analyzer
- **Purpose:** Primary skill builder for evaluating alternative investment options
- **Format:** Interactive web application
- **HTML File:** Investment_Strategy_Comparison_Tool.html

---

## Day 2 Activity Assets

### 2. Real Estate vs REIT Comparator
- **Purpose:** Compare direct property ownership vs. REIT investment with 10-year calculations and trade-off analysis
- **Format:** Interactive calculator/comparison tool
- **HTML File:** Real_Estate_REIT_Comparator.html
- **Supports Activity:** Activity 1 - Real Estate Investment Analysis

### 3. Cryptocurrency Portfolio Simulator
- **Purpose:** Explore cryptocurrency fundamentals and model allocation scenarios with risk assessment
- **Format:** Interactive simulation
- **HTML File:** Crypto_Portfolio_Simulator.html
- **Supports Activity:** Activity 2 - Cryptocurrency Exploration & Risk Assessment

### 4. Alternative Asset Diversification Analyzer
- **Purpose:** Evaluate how adding alternatives (REITs, gold, commodities) affects portfolio characteristics with 30-year projections
- **Format:** Interactive calculator/analyzer
- **HTML File:** Alternative_Asset_Diversification_Analyzer.html
- **Supports Activity:** Activity 3 - Alternative Asset Diversification Analysis

### 5. Alternative Investment Decision Framework
- **Purpose:** Evaluate real alternative investment opportunities (private real estate, gold, leveraged ETFs) using systematic framework
- **Format:** Interactive evaluation tool
- **HTML File:** Alternative_Investment_Decision_Framework.html
- **Supports Activity:** Activity 4 - Alternative Investment Decision Framework Application

### 6. Personal Alternative Investment Strategy Builder
- **Purpose:** Create personal approach to alternatives based on self-assessment and management plan
- **Format:** Interactive planning tool
- **HTML File:** Personal_Alternative_Strategy_Builder.html
- **Supports Activity:** Activity 5 - Personal Alternative Investment Strategy
