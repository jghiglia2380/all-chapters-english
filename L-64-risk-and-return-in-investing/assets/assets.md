# Asset Specifications for L-64: Risk and Return in Investing

## Overview
This chapter includes 7 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Risk-Return Investment Analyzer
- **Purpose:** Primary skill builder for analyzing risk-return tradeoffs in investments
- **Format:** Interactive web application
- **HTML File:** Risk_Return_Analyzer.html

---

## Day 2 Activity Assets

### 2. Risk Tolerance Assessment Tool
- **Purpose:** Determine personal risk tolerance through time horizon, financial situation, and emotional temperament questions
- **Format:** Interactive assessment/quiz
- **HTML File:** Risk_Tolerance_Assessment.html
- **Supports Activity:** Activity 1 - Risk Tolerance Assessment and Analysis

### 3. Risk Metrics Calculator
- **Purpose:** Calculate and interpret standard deviation, beta, and Sharpe ratio for six different investments
- **Format:** Interactive calculator
- **HTML File:** Risk_Metrics_Calculator.html
- **Supports Activity:** Activity 2 - Investment Risk Metrics Calculator

### 4. Portfolio Constructor for Risk Profiles
- **Purpose:** Build diversified portfolios for three investor profiles (Maria age 25, David age 45, Sharon age 63) with metrics analysis
- **Format:** Interactive portfolio builder
- **HTML File:** Portfolio_Constructor_Risk_Profiles.html
- **Supports Activity:** Activity 3 - Portfolio Construction for Different Risk Profiles

### 5. Historical Volatility Analyzer
- **Purpose:** Analyze 30-year historical market data and make investment decisions during simulated scenarios
- **Format:** Interactive simulation with decision points
- **HTML File:** Historical_Volatility_Analyzer.html
- **Supports Activity:** Activity 4 - Historical Volatility Analysis and Decision-Making

---

## Supporting Materials

### 6. Reference Sheet
- **Purpose:** Quick reference for risk metrics, formulas, and key concepts

- **HTML File:** Reference_Sheet.html

### 7. Supporting Worksheet
- **Purpose:** Practice exercises for risk metric calculations and portfolio analysis
