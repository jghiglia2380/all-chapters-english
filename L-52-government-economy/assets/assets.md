# Asset Specifications for L-52: Government and the Economy - Personal Impact

## Overview
This chapter includes 7 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Government Services Analyzer
- **Purpose:** Analyze government services value and personal tax burden with cost-benefit framework
- **Format:** Interactive web application
- **HTML File:** Government_Services_Analyzer.html

---

## Day 2 Activity Assets

### 2. Tax Burden Calculator
- **Purpose:** Calculate total tax burden from federal and state taxes including income tax, Social Security, Medicare, sales tax, property tax, and gas tax
- **Format:** Interactive calculator with burden analysis
- **HTML File:** Tax_Burden_Calculator.html
- **Supports Activity:** Activity 1, Part A - Tax Burden Analysis (8 minutes)

### 3. Government Services Value Estimator
- **Purpose:** Estimate value of government services consumed including public schools, police, fire, roads, parks, libraries, and university subsidies
- **Format:** Interactive value calculator with net contributor analysis
- **HTML File:** Services_Value_Estimator.html
- **Supports Activity:** Activity 1, Part B - Government Services You Use (7 minutes)

### 4. Policy Cost-Benefit Framework
- **Purpose:** Systematically evaluate proposed policies by identifying benefits, costs, unintended consequences, alternatives, and making recommendations
- **Format:** Interactive policy analysis worksheet
- **HTML File:** Policy_Cost_Benefit_Framework.html
- **Supports Activity:** Activity 2 - Policy Cost-Benefit Analysis (15 minutes)

### 5. Regulation Analysis Worksheet
- **Purpose:** Analyze specific state regulations in detail including requirements, intended problems, costs to consumers/businesses/society, and reform recommendations
- **Format:** Interactive regulation analysis tool
- **HTML File:** Regulation_Analysis_Worksheet.html
- **Supports Activity:** Activity 3 - Regulation Deep Dive (10 minutes)

### 6. Policy Impact Analyzer Tool
- **Purpose:** Six-step framework to analyze any government policy: classify type, identify market failure, map winners/losers, assess effectiveness, consider opportunity cost, and make recommendations
- **Format:** Interactive multi-step analysis tool
- **HTML File:** Policy_Impact_Analyzer.html
- **Supports Activity:** Activity 4 - Policy Impact Analyzer Tool (10 minutes)

### 7. Government Economy Reference Sheet
- **Purpose:** Quick reference for market failures, government intervention types, and tax structures
- **Format:** Printable reference document
- **HTML File:** Reference_Sheet.html
- **Supports Activity:** All activities - provides reference information
