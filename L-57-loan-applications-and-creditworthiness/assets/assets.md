# Asset Specifications for L-57: Loan Applications and Creditworthiness

## Overview
This chapter includes 7 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Loan Application Readiness Checker
- **Purpose:** Evaluate loan application strength and identify areas for improvement using Five C's framework
- **Format:** Interactive web application
- **HTML File:** Loan_Application_Readiness_Checker.html

---

## Day 2 Activity Assets

### 2. Documentation Checklist Builder
- **Purpose:** Identify and organize required documentation for different loan types (auto, personal, mortgage)
- **Format:** Interactive checklist tool
- **HTML File:** Documentation_Checklist_Builder.html
- **Supports Activity:** Activity 1 - Documentation Hunt Challenge

### 3. Loan Application Simulator
- **Purpose:** Complete full loan applications with real-time feedback, strength scoring, and approval likelihood
- **Format:** Interactive application simulator
- **HTML File:** Loan_Application_Simulator.html
- **Supports Activity:** Activity 2 - Application Completion Workshop

### 4. Application Strength Analyzer (Five C's)
- **Purpose:** Evaluate borrower profiles using Five C's framework (Character, Capacity, Capital, Collateral, Conditions)
- **Format:** Interactive evaluation tool
- **HTML File:** Application_Strength_Analyzer.html
- **Supports Activity:** Activity 3 - The Five C's Evaluation Station

### 5. Creditworthiness Improvement Planner
- **Purpose:** Assess current financial situation and create 6-12 month improvement plans for better loan terms
- **Format:** Interactive planning tool
- **HTML File:** Creditworthiness_Improvement_Planner.html
- **Supports Activity:** Activity 4 - Creditworthiness Improvement Planning

### 6. Reference Sheet
- **Purpose:** Quick reference for Five C's definitions, DTI/LTV formulas, credit score ranges, documentation requirements
- **Format:** 1-page printable
- **HTML File:** Reference_Sheet.html

### 7. Supporting Worksheet
- **Purpose:** Loan application practice scenarios, creditworthiness self-assessment, documentation checklists
- **Format:** 2-page printable
- **HTML File:** Supporting_Worksheet.html
