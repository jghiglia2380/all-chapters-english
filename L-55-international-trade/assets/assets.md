# Asset Specifications for L-55: International Trade and Economic Development

## Overview
This chapter includes 8 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Currency Exchange Simulator
- **Purpose:** Calculate how exchange rates affect travel, purchases, and trade decisions
- **Format:** Interactive web application
- **HTML File:** Currency_Exchange_Simulator.html

---

## Day 2 Activity Assets

### 2. State Export/Import Analyzer
- **Purpose:** Analyze state exports, import dependence, and personal trade impact calculations
- **Format:** Interactive data analysis tool
- **HTML File:** State_Trade_Analyzer.html
- **Supports Activity:** Activity 1 - {{STATE_NAME}} in the Global Economy

### 3. Supply Chain Mapper
- **Purpose:** Map and visualize global supply chains for common products, identify vulnerability points
- **Format:** Interactive visualization tool
- **HTML File:** Supply_Chain_Mapper.html
- **Supports Activity:** Activity 2 - Global Supply Chain Analysis

### 4. Career Globalization Assessment
- **Purpose:** Assess job vulnerability to globalization and develop career strategies for the global economy
- **Format:** Interactive assessment/planner
- **HTML File:** Career_Globalization_Assessment.html
- **Supports Activity:** Activity 3 - Career Planning in Global Economy

### 5. Trade Impact Analyzer
- **Purpose:** Simulate trade policy decisions (tariffs, free trade) and analyze outcomes for different stakeholders
- **Format:** Interactive simulation
- **HTML File:** Trade_Impact_Analyzer.html
- **Supports Activity:** Activity 4 - Trade Impact Analyzer (Skill Builder)

### 6. Consumer Strategy Planner
- **Purpose:** Develop personal consumption strategies balancing price, domestic support, and ethics
- **Format:** Interactive planning tool
- **HTML File:** Consumer_Strategy_Planner.html
- **Supports Activity:** Activity 5 - Consumer Strategy in Global Economy

### 7. Reference Sheet
- **Purpose:** Quick reference for international trade concepts
- **Format:** 1-page printable
- **HTML File:** Reference_Sheet.html

### 8. Supporting Worksheet
- **Purpose:** Practice exercises for trade analysis
- **Format:** 2-page printable
- **HTML File:** Supporting_Worksheet.html
