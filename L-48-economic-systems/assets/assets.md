# Asset Specifications for L-48: Economic Systems and Resource Allocation

## Overview
This chapter includes 6 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Economic Systems Comparison Tool
- **Purpose:** Comprehensive tool for exploring and comparing economic systems (capitalism, socialism, mixed) across multiple dimensions with real country examples
- **Format:** Interactive web application
- **HTML File:** Economic_Systems_Comparison.html

---

## Day 2 Activity Assets

### 2. Property Rights Impact Simulator
- **Purpose:** Explore how property rights affect investment decisions through business startup and home buying scenarios in strong vs. weak property rights environments
- **Format:** Interactive scenario simulator
- **HTML File:** Property_Rights_Impact_Simulator.html
- **Supports Activity:** Activity 2 - Property Rights Impact Analysis (12 minutes)

### 3. Freedom vs. Security Trade-off Analyzer
- **Purpose:** Analyze trade-offs between economic freedom and security across different systems showing career options, income volatility, safety nets, and retirement projections
- **Format:** Interactive comparison tool with financial projections
- **HTML File:** Freedom_Security_Tradeoff_Analyzer.html
- **Supports Activity:** Activity 3 - Freedom vs. Security Trade-off Evaluation (13 minutes)

### 4. Mixed Economy Designer
- **Purpose:** Design ideal mixed economy by setting business regulation, tax rates, safety net levels, and property rights protection to see projected outcomes
- **Format:** Interactive parameter-based simulator
- **HTML File:** Mixed_Economy_Designer.html
- **Supports Activity:** Activity 4 - Mixed Economy Design Challenge (10 minutes)

### 5. Personal Financial Strategy Worksheet
- **Purpose:** Develop financial strategies appropriate for different economic systems (market, command, mixed) with specific priorities based on system type
- **Format:** Interactive worksheet with recommendations
- **HTML File:** Financial_Strategy_Worksheet.html
- **Supports Activity:** Activity 5 - Personal Financial Strategy Based on System Type (5 minutes)

### 6. Economic Systems Reference Sheet
- **Purpose:** Quick reference for key economic systems concepts, characteristics, property rights impact, and country examples
- **Format:** Printable reference document
- **HTML File:** Economic_Systems_Reference_Sheet.html
- **Supports Activity:** All activities - provides reference information
