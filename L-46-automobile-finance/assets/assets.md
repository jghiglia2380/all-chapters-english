# Asset Specifications for L-46: Automobile Finance - Buying vs. Leasing

## Overview
This chapter includes 5 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Auto Finance Decision Calculator
- **Purpose:** Interactive tool for comprehensive buy vs. lease comparison with state-specific data
- **Format:** Interactive web application
- **HTML File:** Auto_Finance_Decision_Calculator.html

---

## Day 2 Activity Assets

### 2. Total Cost of Ownership Worksheet
- **Purpose:** Calculate TRUE total cost of ownership beyond the sticker price including sales tax, registration, insurance, fuel, maintenance, repairs, and residual value
- **Format:** Interactive calculator with printable results
- **HTML File:** Total_Cost_Ownership_Worksheet.html
- **Supports Activity:** Activity 1 - Real-World Vehicle Cost Breakdown (15 minutes)

### 3. Loan Terms Analyzer
- **Purpose:** Compare loan structures (36, 60, 72 months) showing monthly payments, total interest, equity position, and underwater periods
- **Format:** Interactive calculator
- **HTML File:** Loan_Terms_Analyzer.html
- **Supports Activity:** Activity 3 - Loan Terms Impact Analysis (15 minutes)

### 4. Vehicle Financing Decision Matrix
- **Purpose:** Personalized framework for making financing decisions based on priorities, circumstances, and state-specific factors
- **Format:** Interactive worksheet with printable PDF
- **HTML File:** Vehicle_Financing_Decision_Matrix.html
- **Supports Activity:** Activity 4 - Personal Vehicle Financing Strategy (5 minutes)

### 5. State Cost Reference Sheet
- **Purpose:** Quick reference for all state-specific costs including sales tax, registration fees, insurance averages, gas prices, and loan rates
- **Format:** Auto-populated reference document
- **HTML File:** State_Cost_Reference_Sheet.html
- **Supports Activity:** All activities - provides state-specific data
