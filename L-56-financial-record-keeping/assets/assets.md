# Asset Specifications for L-56: Financial Record Keeping and Account Reconciliation

## Overview
This chapter includes 8 HTML assets for the skill builder and Day 2 activities.

---

## Day 1 Skill Builder

### 1. Checkbook Reconciliation Simulator
- **Purpose:** Practice maintaining accurate financial records and identifying discrepancies through reconciliation
- **Format:** Interactive web application
- **HTML File:** Digital_Filing_System_Organizer.html

---

## Day 2 Activity Assets

### 2. Statement Reconciliation Challenge Tool
- **Purpose:** Complete reconciliation process with realistic monthly scenario including missing transactions
- **Format:** Interactive reconciliation simulator
- **HTML File:** Statement_Reconciliation_Challenge.html
- **Supports Activity:** Activity 1 - Statement Reconciliation Challenge

### 3. Error Detection Workshop Tool
- **Purpose:** Identify common financial record-keeping errors (math mistakes, missing transactions, duplicates, timing)
- **Format:** Interactive error detection practice
- **HTML File:** Error_Detection_Workshop.html
- **Supports Activity:** Activity 2 - Error Detection Workshop

### 4. Fraud Alert Simulator
- **Purpose:** Practice identifying unauthorized transactions and responding appropriately with fraud detection protocols
- **Format:** Interactive fraud detection simulation
- **HTML File:** Fraud_Alert_Simulator.html
- **Supports Activity:** Activity 3 - Fraud Alert Simulation

### 5. Financial Tracking System Designer
- **Purpose:** Design personalized financial record-keeping system choosing between register, spreadsheet, or app methods
- **Format:** Interactive planning tool
- **HTML File:** Financial_Tracking_Designer.html
- **Supports Activity:** Activity 4 - Financial Tracking System Design

### 6. Real-World Scenario Resolver
- **Purpose:** Apply record-keeping skills to solve realistic financial challenges (overdrafts, disputed charges, tax prep)
- **Format:** Interactive scenario-based practice
- **HTML File:** Real_World_Scenario_Resolver.html
- **Supports Activity:** Activity 5 - Real-World Scenario Resolution

### 7. Reference Sheet
- **Purpose:** Quick reference for record-keeping terms, retention periods, reconciliation steps, fraud detection
- **Format:** 1-page printable
- **HTML File:** Reference_Sheet.html

### 8. Supporting Worksheet
- **Purpose:** Transaction recording practice, reconciliation calculations, error detection scenarios
- **Format:** 2-page printable
- **HTML File:** Supporting_Worksheet.html
